Antecedentes
El candidato debe ser capaz de crear una base de datos (MySQL) y montar el entorno para poder desarrollar una web
utilizando HTML, PHP, MySQL y CSS.

Objetivo
Se debe crear una página web donde mostrar un listado de títulos de películas. Al hacer un click sobre un elemento
deberá aparecer más información: rating, fecha e imagen en alguna zona de la página.

Requisitos
Para acceder a la información se debe utilizar dos consultas a la base de datos distintas, una que únicamente recoja
los datos del listado (ejecutada inicialmente) y otra para obtener los datos del detalle que se ejecutará al hacer
click sobre un elemento (mediante una consulta asíncrona).

Material
En la carpeta db_dump está ubicado el archivo 'films.sql' con un script para crear la base de datos.

Entrega
Código fuente completo para poder revisarlo.

¿Qué se valora?
Cumplimiento de los puntos anteriores.
Estructuración del código, comentarios, uso de MVC. (https://pear.php.net/manual/en/standards.php).
Maquetación y CSS (el proyecto incluye librerías de Bootstrap). (http://getbootstrap.com/).
Uso de JQuery (incluido en el proyecto). (https://jquery.com/).

Tiempo estimado de la prueba: 1-3h.
